# Aria
## Project Overview
ARIA is a front-end music web application that allows users to search for songs, create playlists, manage profile information, and interact with music data using the iTunes API.

This project was developed using:
- HTML5
- CSS3
- JavaScript
- localStorage
- iTunes Search API

The purpose of the project was to practice:
- JavaScript functionality
- API integration
- Dynamic UI updates
- localStorage data handling
- Multi-page web application design

---

# Features

## User Authentication
- User login page
- User registration page
- Simulated user session handling
- Logout functionality

## Music Search
- Search for songs and artists
- Retrieve real music data from the iTunes API
- Display song artwork dynamically
- Play song previews

## Playlist Management
- Create playlists
- Add songs to playlists
- Remove songs from playlists
- Dynamically update playlist content

## User Profile
- Store user information
- Update profile details
- Upload profile picture
- Save user data using localStorage

## User Interface
- Responsive layout
- Consistent navigation bar
- Glassmorphism-inspired design
- Gradient background styling
- Dynamic content rendering
